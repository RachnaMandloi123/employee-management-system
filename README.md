# Employee Management System

## Project Overview
This is a backend application built using Spring Boot that manages employee data with secure authentication and role-based access control.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- REST API

## Features
- User Login with JWT Authentication
- Role-Based Authorization (ADMIN / USER)
- Add Employee
- Get All Employees
- Delete Employee (Admin only)
- Exception Handling and Validation

## API Endpoints
- POST /employee/login
- POST /employee/save
- GET /employee/all
- DELETE /employee/delete/{id}

## Architecture
- Controller Layer
- Service Layer
- Repository Layer

## Tools Used
- Postman
- Eclipse IDE
